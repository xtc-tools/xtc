#### Install marimo: 
Marimo is a project dependency but if you want to install it you can either do:
```shell
pip install marimo
```
Or if you use uv:
```shell
uv add marimo
```

#### To start a marimo session:
```shell
marimo run path/to/notebook.py
```

#### To edit the marimo notebook itself you can use:
```shell
marimo edit path/to/notebook.py
```
The documentation of [marimo](https://docs.marimo.io/) : https://docs.marimo.io/
